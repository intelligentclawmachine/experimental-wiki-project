# Robin Dunbar

British anthropologist. Proposed the social brain hypothesis and Dunbar's number (~150) in the 1990s.

## Key Contributions

- **Social brain hypothesis**: primates evolved large brains for social complexity, not just tool use
- **Dunbar's number (~150)**: the cognitive limit on stable social relationships
- **Layered intimacy circles**: social networks scale by ~3x per layer (5 → 15 → 50 → 150 → 500 → 1,500)
- **Relationship maintenance as zero-sum**: investing more in one layer means less for others
- Demonstrated that social network size correlates with default mode neural network size (12+ neuroimaging studies)
